Deep Learning (DL)

Deep learning is structured to mimic the human brain. Enabling AI to find patterns in unstructured data. While Deep learning is a branch of Machine Learning, it is capable of more complex tasks, and most AIs that people think of use Deep Learning.

Neurons



In a Artifial Neural Network, layers of interconnected nodes are called Neurons, and these neurons work together to process and learn from given data.

Layers



Layers in a Neural Network are each designed to find a different feature from the given data set

- A Feature being an attribute associated with the data

Deep Learning uses multiple layers (hundreds and possibly even more) to process data allowing for the completion of more complex tasks

    - Examples of deep learning Algorithms -

  •   Convolutional Neural Networks (CNNs)

  •   Recurrent Neural Networks (RNNs)

  •   Long Short-Term Memory Networks (LSTMs)

  •   Generative Adversarial Networks (GANs)

NOTE:


  Deep Learning struggles to find patterns in new data that is given to it, which is a major flaw as it limits the AI from dealing with new unforseen data after being trained
