Machine Learning (ML)

Machine Learning is a branch of AI learning that involves finding patterns in datasets and learning from those patterns.

  1. The model

- Define the problem
- Gather relevant data

- Process data if needed

  2. Algorithm

- Decide which ML Algorithm you should use

- Each alogrithm has its pros and cons and each can be used for different situations

  3. Data Splitting

- Split your data into different sets

- This way you can Train, Test, and Evaluate the accuracy of your model

  - Examples of machine learning Algorithms -

  •   K Nearest Neighbor

  •   Decision Trees

  •   Linear Regression

  •   Support Vector Machines

  NOTE:

  - The reason why some AIs are biased can be due to the data they are trained with, such as data that has the pontential to be biased will cause the model to be bias when it makes its own predictions or classifications, etc.

  - The quantity and quality of the data as well as the diversity of it can help reduce pontential bias
